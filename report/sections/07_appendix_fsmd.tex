\section{Supplementary FSMD Views}

The following control-state diagrams are intentionally simplified. They do not
attempt to reproduce every temporary register or default assignment in the RTL.
Instead, they summarize the dominant control flow behind the configuration path
so the higher-level block diagrams in the main text remain focused on the data
movement.
